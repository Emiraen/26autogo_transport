# 舵机驱动说明文档

本文档介绍 `chassis/include/servo.h` 与 `chassis/src/servo.c` 的设计与使用方法，目标读者是希望快速理解和调用舵机模块的同学，不要求 STM32 HAL 经验。

---

## 1. 模块概览

本模块基于 **STM32F1 HAL 库**，使用定时器输出 PWM 控制 3 个 **DS3225 数字舵机**：

| 舵机用途       | 宏定义 ID          | 引脚 | 定时器通道   |
| -------------- | ------------------ | ---- | ------------ |
| 物料盘 (转盘)  | `SERVO_ID_PLATE`   | PA7  | TIM3_CH2     |
| 塔吊底座       | `SERVO_ID_BASE`    | PA6  | TIM3_CH1     |
| 爪子 (夹取)    | `SERVO_ID_CLAW`    | PA8  | TIM1_CH1     |

### 1.1 舵机 PWM 基础

DS3225 这类标准数字舵机要求：

- **PWM 周期**: 20 ms（50 Hz）
- **脉宽范围**: 500 µs ~ 2500 µs
- **角度范围**: 0° ~ 270°（即每度约 7.41 µs）

公式：

```
脉宽(µs) = 500 + 角度(°) × (2500 - 500) / 270
```

### 1.2 时钟配置

- **TIM3** 挂在 APB1，倍频后 **72 MHz**；**TIM1** 挂在 APB2，**72 MHz**。
- 两个定时器都设置 `PSC = 71`，得到 **1 MHz** 计数节拍 (1 个 tick = 1 µs)。
- `ARR = 19999`，对应 20 ms 周期。
- 之后 **CCR 寄存器的值就直接等于脉宽 µs**，编程非常直观。
- TIM1 是高级定时器，需要额外调用 `__HAL_TIM_MOE_ENABLE` 打开主输出。

### 2.1 关键常量（servo.h）

| 宏                       | 值              | 含义                |
| ------------------------ | --------------- | ------------------- |
| `SERVO_PWM_FREQ_HZ`      | 50              | PWM 频率            |
| `SERVO_PULSE_MIN_US`     | 500             | 0° 对应脉宽         |
| `SERVO_PULSE_MAX_US`     | 2500            | 270° 对应脉宽       |
| `SERVO_ANGLE_MAX_DEG`    | 270             | 最大角度            |
| `PLATE_SLOT_COUNT`       | 3               | 物料盘工位数        |
| `PLATE_STEP_DEG`         | 120             | 每工位步进角度      |
| `BASE_HOME_DEG`          | 0               | 塔吊底座初始位置    |
| `BASE_WORK_DEG`          | 90              | 塔吊底座工作位置    |
| `CLAW_OPEN_DEG_DEFAULT`  | 30              | 爪子默认张开角      |
| `CLAW_CLOSE_DEG_DEFAULT` | 120             | 爪子默认闭合角      |

## 3. API 速查

### 3.1 初始化

```c
void Servo_Init(void);
```

- 配置 PA6/PA7/PA8 为复用推挽输出。
- 初始化 TIM3（CH1+CH2）和 TIM1（CH1），启动 PWM。
- 把 3 路舵机置于安全初始位：
  - 底座 → 0°
  - 物料盘 → 0°（工位 0）
  - 爪子 → 张开

> ⚠️ `Servo_Init()` 必须在 `HAL_Init()` 与系统时钟配置之后调用。

### 3.2 通用控制

```c
bool  Servo_SetAngle  (ServoId id, float deg);     // 角度 0~270
bool  Servo_SetPulseUs(ServoId id, uint16_t pulse);// 脉宽 500~2500 µs
float Servo_GetAngle  (ServoId id);                // 读取缓存角度
```

- 越界自动钳位，不会出错。
- 设置脉宽时会反推角度并存到缓存里，`Servo_GetAngle()` 拿到的是最近一次写入对应的角度。
- **注意**：这只是"我指令了什么"，不是真实测量值（DS3225 没有反馈）。

### 3.3 物料盘（转盘）

物料盘有 3 个工位，对应角度 0°、120°、240°。

```c
void    Servo_PlateGotoSlot(uint8_t slot_index); // 跳到指定工位
void    Servo_PlateNextSlot(bool forward);       // 顺/逆时针转一格
uint8_t Servo_PlateGetSlot(void);                // 当前工位 0/1/2
```

示例：每次按键转一个工位

```c
Servo_PlateNextSlot(true);   // 顺时针 120°
```

### 3.4 塔吊底座

```c
void Servo_BaseSetWorking(bool working);
// working = true  → 摆到 BASE_WORK_DEG (90°)
// working = false → 回到 BASE_HOME_DEG (0°)
```

### 3.5 爪子

```c
void Servo_SetClawAngles(float open_deg, float close_deg); // 自定义开合角
void Servo_ClawSet(bool close);                            // true=合, false=开
bool Servo_ClawIsClosed(void);                             // 当前是否闭合
```

如果默认 30°/120° 不合适，可在初始化后调一次 `Servo_SetClawAngles` 修改。

## 4. 使用示例

```c
#include "servo.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();

    Servo_Init();                        // 初始化 3 路舵机

    Servo_SetClawAngles(20.0f, 110.0f);  // 调整爪子开合角

    while (1)
    {
        Servo_BaseSetWorking(true);      // 塔吊伸出
        HAL_Delay(800);

        Servo_ClawSet(true);             // 闭合抓物
        HAL_Delay(500);

        Servo_BaseSetWorking(false);     // 塔吊缩回
        HAL_Delay(800);

        Servo_PlateNextSlot(true);       // 物料盘换下一工位
        HAL_Delay(1000);

        Servo_ClawSet(false);            // 张开放料
        HAL_Delay(500);
    }
}
```

## 5. 常见问题（FAQ）

**Q1：舵机不动 / 抖动？**
- 检查供电：DS3225 工作电流较大（堵转可达 2~3 A），请用独立 5~7.4 V 大电流电源，**信号地与 STM32 共地**。
- 信号线插对 PA6/PA7/PA8。
- 如果只有 TIM1（爪子）不动，多半是忘了 `__HAL_TIM_MOE_ENABLE`，本驱动里已经打开。

**Q2：角度不准、走得不到位？**
- DS3225 出厂个体差异，可在 `Servo_SetClawAngles` 之类接口里微调。
- 物料盘工位对不齐：以机械零位重新标定 0°，或修改 `PLATE_STEP_DEG`。

**Q3：可以加新舵机吗？**
1. 在 `ServoId` 枚举里添加新 ID（在 `SERVO_ID_COUNT` 之前）。
2. 在 `servo_gpio_init` 里加上对应 GPIO，在合适的定时器里再 `HAL_TIM_PWM_ConfigChannel + Start`。
3. 在 `Servo_SetPulseUs` 的 `switch` 里加一条新分支调用 `__HAL_TIM_SET_COMPARE`。

**Q4：怎么换不同型号的舵机（180°/360°）？**
- 改 `SERVO_PULSE_MIN_US`、`SERVO_PULSE_MAX_US`、`SERVO_ANGLE_MAX_DEG` 三个宏，其它代码不用动。

**Q5：能让舵机平滑转动吗？**
当前驱动是"瞬时跳变"模式（直接改 CCR）。如需平滑，可在上层用定时器/任务每隔几毫秒调用 `Servo_SetAngle` 做一段斜坡，例如：

```c
void Servo_RampTo(ServoId id, float target, float step_deg, uint32_t step_ms)
{
    float cur = Servo_GetAngle(id);
    while (fabsf(target - cur) > step_deg) {
        cur += (target > cur) ? step_deg : -step_deg;
        Servo_SetAngle(id, cur);
        HAL_Delay(step_ms);
    }
    Servo_SetAngle(id, target);
}
```

## 6. 设计要点小结

1. **统一 1 µs tick**：让脉宽与 CCR 直接相等，无需复杂换算。
2. **角度优先 API**：业务层只关心角度，底层处理脉宽与钳位。
3. **业务封装**：物料盘/底座/爪子各自有语义化函数，避免到处写魔法数字。
4. **状态缓存**：维护工位、爪子开/合状态，便于上层做"下一格""切换状态"等逻辑。
